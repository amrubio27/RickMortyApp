> **ES**: Proyecto de estudio para **Kotlin Multiplatform (KMP)** y **Compose Multiplatform (CMP)**, usando el universo de Rick & Morty para experimentar con UI compartida y lógica común.  
> **EN**: Study project for **Kotlin Multiplatform (KMP)** and **Compose Multiplatform (CMP)**, using the Rick & Morty universe to experiment with shared UI and common logic.

---

## ✨ Características / Highlights

### 🔁 Kotlin Multiplatform (KMP)
- **Lógica compartida** entre Android e iOS.
- Reutilización de modelos, casos de uso y capa de datos.

### 🎨 Compose Multiplatform (CMP)
- UI declarativa **compartida** en múltiples plataformas.
- Componentes reutilizables y estado predecible.

### 🧪 Proyecto de aprendizaje / Learning Project
- Enfoque en **arquitectura multiplataforma** real.
- Ideal para validar patrones y buenas prácticas.
